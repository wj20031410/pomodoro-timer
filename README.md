# 番茄钟项目

一个简洁美观的番茄工作法计时器应用。

## 功能特性

- 🍅 **番茄计时**：25分钟专注 + 5分钟休息
- ⏱️ **自定义时长**：可调整专注和休息时间
- 🎵 **提示音**：计时结束播放提示音
- 📊 **今日统计**：进度条显示完成情况
- 📈 **本周统计**：柱状图展示每日数据
- 💾 **数据持久化**：自动保存到浏览器本地存储
- 📤 **数据导出/导入**：支持备份和恢复数据

## 技术栈

- HTML5
- CSS3（玻璃态设计）
- JavaScript（纯原生）
- localStorage（数据存储）

## 快速开始

### 方法一：双击启动（推荐）

1. 双击 `会议白板助手.command` 文件
2. 自动打开浏览器访问应用

### 方法二：手动启动

```bash
# 进入代码目录
cd code

# 启动 HTTP 服务器
python3 -m http.server 8080

# 打开浏览器访问
open http://localhost:8080
```
