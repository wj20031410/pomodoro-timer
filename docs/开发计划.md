# 番茄钟项目 - 开发计划

> 本文档供后续 AI 助手使用，每一步可独立执行。
> 开始开发前，请先阅读 `AGENTS.md` 和 `docs/PRDS/PRD.md`。

---

## 项目概述

基于番茄工作法的专注计时器应用，支持自定义时长、今日/本周统计、数据导出导入，采用苹果风格玻璃态设计。

## Step 2：计时器核心逻辑

### 功能描述
实现倒计时逻辑，包括开始、暂停、重置、模式切换（专注/休息）。

### 涉及文件
- `code/scripts/app.js`
- `code/index.html`

### 实现步骤
1. 在 `state` 中定义属性：
   - `mode`: 'focus' | 'break'
   - `status`: 'idle' | 'running' | 'paused'
   - `focusDuration`: 25（分钟）
   - `breakDuration`: 5（分钟）
   - `remainingSeconds`: 倒计时秒数
2. 实现函数：
   - `startTimer()`: 开始倒计时
   - `pauseTimer()`: 暂停倒计时
   - `resetTimer()`: 重置到初始值
   - `tick()`: 每秒回调，减少 `remainingSeconds`
3. 在 HTML 中添加按钮：开始、暂停、重置

### 完成标准
- [ ] 点击「开始」后，倒计时每秒减少
- [ ] 点击「暂停」后，倒计时停止
- [ ] 点击「重置」后，倒计时回到初始值（25:00 或 05:00）
- [ ] 倒计时归零后，自动切换模式（专注→休息 或 休息→专注）

## Step 4：音频提示功能

### 功能描述
计时结束时播放提示音。

### 涉及文件
- `code/scripts/app.js`

### 实现步骤
1. 实现 `initAudio()`：创建 AudioContext（懒加载，用户首次交互后才初始化）
2. 实现 `playBeep(frequency, duration)`：生成指定频率和时长的哔声
3. 实现 `playCompletionSound()`：播放 3 声哔声（800Hz×2 + 1200Hz×1）
4. 在 `tick()` 中，计时归零时调用 `playCompletionSound()`

### 完成标准
- [ ] 计时结束（归零）时播放 3 声哔声
- [ ] 刷新页面后首次提示音仍能播放（需要用户先点击任意按钮）

## Step 5：设置页面与数据持久化

### 功能描述
实现设置页面（自定义专注/休息时长、每日目标），数据保存到 localStorage。

### 涉及文件
- `code/scripts/app.js`
- `code/styles/main.css`
- `code/index.html`

### 实现步骤
1. 在设置视图中添加 3 个输入框：专注时长、休息时长、每日目标
2. 实现 `saveSettings()`：读取输入值，验证范围（1-180/1-60/1-24），更新 `state`
3. 实现 `saveData()`：将 `state` 序列化为 JSON，存入 `localStorage`（key: `pomodoro-data-v1`）
4. 实现 `loadData()`：从 `localStorage` 读取并解析
5. 实现 `initStateFromStorage()`：在应用初始化时加载数据
6. 实现 `renderSettingsView()`：将 `state` 的值填充到输入框

### 完成标准
- [ ] 设置页面显示 3 个输入框，初始值为 25/5/6
- [ ] 点击「保存」后，数据存入 localStorage
- [ ] 刷新页面后，设置值从 localStorage 恢复

## Step 6：数据导出/导入功能

### 功能描述
支持导出 JSON 数据文件和导入备份。

### 涉及文件
- `code/scripts/app.js`

### 实现步骤
1. 实现 `exportData()`：
   - 将 `state` 打包为 JSON 对象
   - 创建 Blob，生成下载链接
   - 自动下载为 `pomodoro-backup-YYYY-MM-DD.json`
2. 实现 `importData(file)`：
   - 使用 FileReader 读取文件内容
   - 解析 JSON，验证数据格式
   - 更新 `state` 并保存到 localStorage
3. 在设置页面添加「导出」「导入」按钮
4. 导入成功后显示提示

### 完成标准
- [ ] 点击「导出」下载 JSON 文件
- [ ] 选择 JSON 文件后，数据恢复并显示「已导入」提示
- [ ] 导入错误格式文件显示「导入失败」提示

## Step 7：Dashboard 页面（统计卡片）

### 功能描述
添加今日统计卡片和本周柱状图。

### 涉及文件
- `code/scripts/app.js`
- `code/styles/main.css`
- `code/index.html`

### 实现步骤
1. 在计时视图中添加两个新卡片：
   - **今日统计卡片**：显示完成数、进度条、目标
   - **本周统计卡片**：显示柱状图（周一至周日）
2. 在 `state` 中添加：
   - `completedPomodoros`: 今日完成数
   - `weekData`: 数组 [周一, 周二, ..., 周日]
3. 实现 `updateWeekPomodoros()`：计时结束时更新本周数据
4. 实现 `initWeekChart()`：初始化 Canvas
5. 实现 `drawWeekChart()`：使用 Canvas 绘制柱状图
   - 柱子宽度 28px，间距 8px
   - 今日对应的柱子用橙色高亮
   - 其他柱子用灰色
6. 实现 `updateWeekTotal()`：计算并显示本周总数
7. 实现 `updateCompletedInfo()`：更新今日统计和柱状图
8. 在 `tick()` 中，完成专注时增加 `completedPomodoros` 并更新柱状图

### 完成标准
- [ ] 今日统计卡片显示完成数和进度条
- [ ] 本周柱状图显示周一到周日的数据
- [ ] 今日柱子用橙色高亮，其他柱子灰色
- [ ] 完成一个番茄钟后，今日统计和本周柱状图都更新

## 技术栈总览

| 类别 | 技术 |
|------|------|
| **前端框架** | 纯 HTML + CSS + JavaScript（零依赖） |
| **样式** | CSS3（CSS 变量、Flexbox、Grid、backdrop-filter） |
| **图表** | Canvas API（原生绘制柱状图） |
| **音频** | Web Audio API（生成提示音） |
| **数据存储** | localStorage（浏览器本地存储） |
| **部署方式** | 静态文件部署（可部署到 GitHub Pages / Netlify / Vercel） |
| **HTTP 服务** | Python http.server（本地开发） |
