# 番茄钟 PRD

## 1. 产品概述

一个简洁美观的番茄工作法计时器，帮助用户保持专注、提升工作效率。

**核心价值**：通过「25 分钟专注 + 5 分钟休息」的节奏训练，帮助用户克服拖延，保持高效工作状态。

---

## 2. 目标用户

- 需要专注工作或学习的上班族、学生
- 想要改善时间管理的人群
- 轻度使用：不需要复杂功能，只需一个好看的计时器

---

## 3. 核心功能清单

### P0 - 必须有（最小可行产品）

| 功能 | 描述 |
|------|------|
| 专注计时 | 25 分钟倒计时，显示 MM:SS 格式 |
| 休息计时 | 5 分钟倒计时，自动切换 |
| 开始/暂停 | 控制计时器运行状态 |
| 重置 | 恢复到初始状态 |
| 提示音 | 计时结束时播放 3 声哔声 |

### P1 - 重要功能

| 功能 | 描述 |
|------|------|
| 自定义时长 | 用户可设置专注时长（1-180 分钟）和休息时长（1-60 分钟） |
| 数据持久化 | 自动保存到浏览器 localStorage，刷新不丢失 |
| 数据导出 | 导出 JSON 备份文件 |
| 数据导入 | 导入 JSON 恢复数据 |

### P2 - 增强功能

| 功能 | 描述 |
|------|------|
| 今日统计 | 显示今日完成数 + 进度条 |
| 本周统计 | 柱状图展示周一至周日完成数 |
| 每日目标 | 可设置每日目标数（1-24 个） |

## 5. 设计风格

### 5.1 配色方案

| 类型 | 颜色 | 用途 |
|------|------|------|
| 页面背景 | `#fafafa` | 主背景，柔和米白色 |
| 卡片背景 | `rgba(255,255,255,0.7)` | 玻璃态效果 |
| 文字主色 | `#1d1d1f` | 标题和主要文字 |
| 文字辅色 | `#86868b` | 次要说明文字 |
| 强调色 | `#ff9500` | 按钮、进度条、高亮 |
| 休息色 | `#007aff` | 休息模式进度圈 |

### 5.2 设计特点

- **苹果风格**：大圆角（24px）、柔和阴影、玻璃态效果
- **极简主义**：减少视觉噪音，专注核心功能
- **触摸友好**：按钮高度 ≥ 48px，触摸有反馈

## 6. 技术方案

- **纯前端**：HTML + CSS + JavaScript，无框架依赖
- **存储**：localStorage
- **音频**：Web Audio API
- **图表**：Canvas API
- **部署**：静态文件（GitHub Pages / Netlify / Vercel）

---

## 7. 交互规则

| 状态 | 行为 |
|------|------|
| 点击「开始」 | 倒计时开始运行，按钮变成「暂停」 |
| 点击「暂停」 | 倒计时停止，按钮变回「开始」 |
| 点击「重置」 | 倒计时回到初始值（25:00 或 05:00） |
| 计时归零 | 播放提示音，切换模式 |
| 计时归零（专注） | `completedPomodoros++`，切换到休息 |
| 计时归零（休息） | 切换到专注 |

## 8. 数据模型

### localStorage Key: `pomodoro-data-v1`

```json
{
  "focusDuration": 25,
  "breakDuration": 5,
  "dailyGoal": 6,
  "completedPomodoros": 4,
  "todayDate": "Wed Jun 11 2026",
  "weekData": [1, 2, 3, 4, 2, 1, 0],
  "exportedAt": "2026-06-11T08:00:00.000Z"
}
```

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| focusDuration | number | 专注时长（分钟） |
| breakDuration | number | 休息时长（分钟） |
| dailyGoal | number | 每日目标数 |
| completedPomodoros | number | 今日完成数 |
| todayDate | string | 今天的日期字符串 |
| weekData | number[] | 本周数据（周一到周日） |
| exportedAt | string | 导出时间（仅导出时有） |
